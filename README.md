# Clerkk

Personal finance dashboard app for Canadian professionals.

## Features
- Auth0 Google OAuth authentication
- Manual income and expense tracking
- Dashboard with financial insights
- FastAPI backend with PostgreSQL
- React Native (Expo) mobile app

## Tech Stack
- **Backend:** FastAPI, SQLAlchemy, PostgreSQL, Auth0
- **Frontend:** React Native, Expo Router, TypeScript
- **DevOps:** Docker, pre-commit hooks (Black + Prettier)

## Setup

### Backend
```bash
cd backend
docker compose up
```

### Frontend
```bash
cd clerkk
npm install
npx expo start
```
